# Peque Lang

Peque is a tiny programming language created in Brazil by Iago Dórea.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     peque:
       github: southernclaim/peque
   ```

2. Run `shards install`

## Usage

```crystal
require "peque"
```
